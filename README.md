# McChicken
Generating summary statistics from broiler video feed for automated chicken welfare monitoring

*Brief breakdown of report:*
1) Chicken detection
   - Local thresholding
   - Mask R-CNN
   - YOLOv5
2) Metrics
   - Occupancy changes
   - Feeder utilization
   - Occupancy heatmaps
   - Network statistics
